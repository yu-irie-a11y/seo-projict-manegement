# Otona Table KW選定

## 評価基準

### ブランド適合性
- **◎ 最適**: 食事・飲み・友達・居場所・メンタルケア寄り（KOKONEコンセプト直結）
- **○ 許容**: 恋愛・セカンドパートナー系（入口として使用、記事内で調整必要）
- **△ 注意**: 恋愛HOW TO系（境界線・リスク・安全な楽しみ方をセットで書く）
- **▲ 要検討**: ネガティブ受け皿型（ネガ→安心→サービス紹介の流れ）
- **× NG**: 不倫・肉体関係を推奨する表現

### 先方FBのポイント
1. 恋愛寄りKWを入口にする発想自体は妥当
2. 記事の中身と導線は「安心・食事・友達・心の拠り所」に寄せる
3. ネガティブ系＆恋愛寄りKW比率がやや高い → ポジティブ寄りを厚めに

## 記事設計の方針

### 二段構えの設計
1. **入口（流入KW）**: 恋愛・ネガ寄りも許容
2. **記事内容・導線**: 「安心・食事・友達・心の拠り所」に寄せる

### 恋愛系KWの注意点
記事内で必ず含める要素：
- 境界線の引き方
- リスクや注意点
- 安全な距離感の保ち方
- 食事・会話中心ならリスクコントロールできる

### ネガティブ系KWの設計
「ネガティブ検索 → 安心感 → サービス紹介」の流れ
- 怖い話だけで終わらず、安全な出口を提示
- 「食事・友達系」のカテゴリ分けを行い、Otona Tableの立ち位置を明確化
