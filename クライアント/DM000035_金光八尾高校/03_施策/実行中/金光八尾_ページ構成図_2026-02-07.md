# 金光八尾中学校・高等学校 ページ構成図

## 概要

- **対象**: 全72ページ（11テンプレート）
- **目的**: 各ページのセクション構成とリンク遷移先を視覚化し、デザイナーへの指示書とする
- **対応資料**: 金光八尾_リンク設計書_2026-02-07.md（`#N` はリンク設計書の番号に対応）
- **作成日**: 2026年2月7日

## 凡例

```
  ページ（上から下にスクロール）                    遷移先
┌──────────────────────────────────────┐
│ セクション名                          │
│   🖼 画像                             │
│   【ボタン】 ← CTAボタン              │──→ 📌 CTA遷移先（赤）    #N
│   テキストリンク ＞                    │──→ 内部リンク遷移先      #N
│   外部リンク ＞                        │──→ 🟣 外部サイト（紫）   #N
│   📎 PDF                             │──→ 🟢 PDFダウンロード（緑）#N
├──────────────────────────────────────┤
│ 次のセクション                         │
│   ...                                │
└──────────────────────────────────────┘

記号:  【 】= CTAボタン   ＞ = テキストリンク   🖼 = 画像
      📌 = CV先(赤)   🟣 = 外部(紫)   🟢 = PDF(緑)
      #N = リンク設計書の番号に対応
```
