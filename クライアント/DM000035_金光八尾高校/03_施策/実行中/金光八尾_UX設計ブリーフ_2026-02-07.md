# 金光八尾中学校・高等学校 UX設計ブリーフ

## 本資料の目的

デザイン着手前に、以下を確定させるための整理資料。

1. **誰が**このサイトを使うのか
2. **何を達成してもらうのか**（コンバージョン）
3. **どう動いてもらうのか**（ユーザーフロー）
4. **各ページは何のために存在するのか**（ページ役割）

> この内容をお客様と合意した時点で、デザイン着手のGOサインとする。

---

## 1. ユーザー定義

### 1-1. ターゲット優先度

ヒアリングより、以下の優先順位で設計する。

| 優先度 | ターゲット | サイトでの主な行動 |
|:---:|---|---|
| **1位** | 中学受験を検討する小5-6保護者 | コース比較 → 説明会予約 / 資料請求 |
| **2位** | 高校受験を検討する中3生とその保護者 | コース比較 → 説明会予約 / 資料請求 |
| **3位** | 在校生の保護者 | お知らせ確認 / 行事予定 / 書類DL |
| （4位） | 卒業生 | 証明書発行 / 同窓会情報 |

> **設計上の判断基準**: 1位・2位のターゲットに最適化する。3位以降は「迷わず辿り着ける」ことを最低条件とし、動線の主役にはしない。

### 1-2. 各ターゲットの関心事と不安

ヒアリングから抽出。**動線設計で「どのページでどの情報を見せるか」の根拠になる。**

#### 第1ターゲット: 中学受験保護者

| 関心事（知りたいこと） | 対応コンテンツ |
|---|---|
| どんなコースがあるか・違いは何か | コース詳細ページ（P17-18） |
| 少人数でどこまで面倒を見てくれるか | 学びの特色（P15）、在校生の声（P60） |
| 塾に通わなくても大丈夫か | 学びの特色（P15）、FAQ（P62）、放課後学習環境 |
| いじめ対策はどうなっているか | FAQ（P62）、学校紹介（P10） |
| 高校進学時にどうなるか（6年一貫の流れ） | 6年間の学び（P31）、コース接続図 |
| 学費はいくらかかるか | 学費（P40）、奨学金 |
| 通学の利便性 | アクセス（P72）、スクールバス（P42） |

| 不安・障壁 | 解消するコンテンツ |
|---|---|
| 私立は学費が高い | 学費ページ + 奨学金情報 + 「塾不要」の訴求 |
| 子どもに合うかわからない | 在校生の声 + 説明会体験 |
| 他校（近大附属・浪速・常翔）と比較中 | 差別化ポイントの明示（少人数・面倒見・自習室） |

#### 第2ターゲット: 高校受験 中3生＋保護者

| 関心事 | 対応コンテンツ |
|---|---|
| コースの選択肢（4コース） | コース詳細（P19-22） |
| 進学実績（大学合格） | 合格実績（P35）、合格体験記（P59） |
| 学校の雰囲気・部活 | 学校生活（P7）、クラブ活動（P28）、クラブブログ（P57） |
| 美術コースの特色 | 美術コース（P22）、ギャラリー（P53） |

#### 第3ターゲット: 在校生保護者

| 関心事 | 対応コンテンツ |
|---|---|
| お知らせの確認 | お知らせ一覧（P54） |
| 行事予定 | 行事予定（P43） |
| スクールバス | バス運行表（P42） |
| 各種書類のダウンロード | 証明書（P45）、各種PDF |

> このターゲットは在校生TOP（P13）をブックマーク利用する想定。トップページからの動線は不要。

### 1-3. 認知経路（ヒアリングより）

サイトへの流入元。**どのページが「最初に見られるか」を決定する根拠。**

| 経路 | 想定される着地ページ | 重要度 |
|---|---|---|
| **塾からの紹介**（最多） | トップページ or コース詳細 | 高 |
| Google検索「金光八尾」 | トップページ | 高 |
| Google検索「金光八尾 コース」「金光八尾 学費」等 | 各詳細ページ直接 | 高 |
| 口コミ・先輩紹介（強化したい） | トップページ or 体験記 | 中 |
| 説明会イベント後の再訪 | トップページ or 入試情報 | 中 |
| SNS（Instagram / YouTube） | トップページ | 低 |

## 2. コンバージョン定義

### 2-1. CV優先度

| 優先度 | CV | 理由 |
|:---:|---|---|
| **1位** | 説明会予約 | 来校が最も入学決定に直結する（ヒアリングより塾紹介→説明会が主要経路） |
| **2位** | 資料請求 | 説明会に来られない層の受け皿。比較検討中の保護者に紙資料を届ける |
| **3位** | イベント予約（オープンスクール等） | 説明会より気軽。特に中学受験層の初接触 |
| **4位** | お問い合わせ | 個別の質問・相談。CV率は低いが検討度は高い |

### 2-2. CVまでのステップ（ファネル）

```
認知                興味                 比較・検討              不安解消             行動
「金光八尾って     「コースが            「近大附属と           「学費は大丈夫？    「説明会に
 どんな学校？」     面白そう」            どっちがいい？」        いじめは？」        行ってみよう」

  トップページ    コース詳細           合格実績              FAQ              説明会予約
  学校紹介TOP    学びの特色           合格体験記            学費              資料請求
                 在校生の声           他校との差別化         在校生の声         お問い合わせ
```

**各ページは、このファネルのどこに位置するかで役割が決まる。**

## 3. ユーザーフロー（推奨5本）

### フロー1: 中学受験保護者の王道動線（最重要）

```
塾からの紹介 or 検索
  → 総合TOP（/）
    → 中学校TOP（/junior/）
      → コース詳細（S特進 or 特進）
        → 在校生の声 or 合格体験記
          → 説明会予約 ★CV
```

**設計ポイント**:
- TOPから中学校TOPへの誘導を最も目立たせる
- コース詳細ページが「比較検討」の中心。ここに在校生の声・体験記への導線を必ず置く
- コース詳細→説明会予約のCTAを強くする（「このコースを体験する」等）

### フロー2: 高校受験生の王道動線

```
検索「金光八尾 高校」
  → 高等学校TOP（/senior/）
    → コース詳細（4コースのいずれか）
      → 合格実績 or 合格体験記
        → 説明会予約 or 資料請求 ★CV
```

**設計ポイント**:
- 高校は4コースあるため、コース選択の分岐がTOP直下で重要
- 「進学実績」が高校受験層の関心No.1。コース詳細から実績ページへの動線を太くする
- 美術コースは独自性が高いので、専用の導線も検討

### フロー3: 不安解消→CV動線

```
検索「金光八尾 学費」「金光八尾 いじめ」等
  → 学費ページ or FAQ（直接着地）
    → 学びの特色（少人数・放課後学習の訴求）
      → 在校生の声（リアルな安心感）
        → 説明会予約 or 資料請求 ★CV
```

**設計ポイント**:
- 学費・FAQは検索から直接着地する可能性が高い
- 不安を解消した直後にCTAを置く（「もっと詳しく知りたい方は説明会へ」）
- 「塾不要」「放課後20時まで自習室」はコスト面の不安解消材料として学費ページにも掲載

### フロー4: イベント駆動型（説明会告知経由）

```
塾のチラシ / SNS告知 / 既存接触者へのメール
  → 説明会・イベント一覧（/admission/event/）
    → 説明会予約 ★CV
```

**設計ポイント**:
- 最短2ステップでCVに至る。途中で迷子にさせない
- イベントページに「何が体験できるか」を明記する（保護者が参加の意思決定に使う）

### フロー5: リピーター動線（説明会参加後の再訪）

```
説明会に参加した後、帰宅して再訪
  → 総合TOP or 入試情報TOP
    → 募集要項 → 学費 → 出願準備
    → 資料請求（紙で手元に残したい）★CV
```

**設計ポイント**:
- 説明会参加者は「入試情報」セクションに直行する
- 募集要項PDF・学費・出願カレンダーを迷わず見つけられることが重要
- 「資料請求」は説明会後の後押しとして機能する
