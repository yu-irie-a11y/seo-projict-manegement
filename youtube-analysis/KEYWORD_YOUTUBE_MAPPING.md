### プロサイトから借用すべき記事構成パターン

プロサイトの記事URL構造から、以下の**記事テンプレートパターン**が見える：

```
【パターン1：評判・口コミ型】（キラーページ向き）
  プロサイト例: rakuten-mobile-reputation/
  構成: 口コミ → メリデメ → 料金 → キャンペーン → 申込手順
  CV設計: 最後の「申込手順」で紹介CPへ誘導

【パターン2：比較型】（検討層の受け皿）
  プロサイト例: rakuten-mobile-ahamo-compare/
  構成: 両者の特徴 → 料金比較 → 速度比較 → どちらが合うか → 楽天を選ぶ場合の申込方法
  CV設計: 「楽天モバイルを選ぶ場合」パートで紹介CPへ

【パターン3：乗り換え型】（CV直結）
  プロサイト例: from-au-norikae/ / from-docomo-norikae/
  構成: 現在の不満点 → 乗り換え先候補 → 楽天モバイルが最適な理由 → 手順 → キャンペーン
  CV設計: 「最もお得な申込方法」として紹介CPを提示

【パターン4：解約型】（他社→楽天への誘導）
  プロサイト例: ahamo-cancel/ / linemo-cancel/
  構成: 解約タイミング → 解約手順 → 解約後のおすすめ → 楽天モバイルの紹介
  CV設計: 「おすすめの乗り換え先」として楽天モバイル＋紹介CPを提示

【パターン5：割引・特典型】（家族CV向き）
  プロサイト例: rakuten-mobile-family/ / rakuten-mobile-gakuwari/
  構成: 割引の概要 → 他社との比較 → 適用条件 → 家族全員で使うとどうなるか → 申込手順
  CV設計: 「家族○人で年間○○円お得」→紹介CP×家族人数のポイント訴求
```
