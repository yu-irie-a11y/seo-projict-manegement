# 運用フローガイド

## このリポジトリで管理するもの
- **提案中案件**: `00_提案中/`
- **案件運用（クライアント別）**: `クライアント/<社名>/`
- **案件別ライター稼働**: `クライアント/<社名>/07_ライター稼働/`
- **案件別収支**: `クライアント/<社名>/収支.md`
- **ライターマスター**: `ライター管理/ライター一覧.md`
- **全社ライター集計**: `ライター管理/月別消化率.md`
- **全社収支（年度別）**: `収支管理/`
- **テンプレート**: `テンプレート/`

## 推奨フロー（提案〜運用〜効果検証）
1. **提案前〜提案後**
   - `00_提案中/README.md` に案件登録
   - `テンプレート/提案中_案件テンプレート.md` を使って要点を整理
2. **受注（運用開始）**
   - `クライアント/<社名>/` を作成（または既存を使用）
   - クライアント`README.md`に **案件概要・KPI・収支（売上/経費/粗利見込み）** を記入
3. **日々の運用**
   - 打ち合わせ: `02_議事録/YYYY/`
   - 施策: `03_施策/実行中` → 完了したら `03_施策/完了`
   - 進捗: `04_進捗/YYYY/`
   - 記事納品: `06_納品物/記事/YYYY/`
4. **効果検証**
   - `05_効果検証/YYYY/` に記録
   - `テンプレート/効果検証レポートテンプレート.md` を利用
5. **ライター稼働管理（随時〜月末）**
   - 発注時: `クライアント/<社名>/07_ライター稼働/YYYY_MM_稼働.md` に記入
   - 随時: 納品・検収に応じてステータス更新
   - 月末: 原価を確定
6. **収支の更新（月末）**
   - 各案件の `07_ライター稼働/` から原価合計を集計
   - `収支管理/YYYY/01_収支.md` に案件別の売上・経費を入力
   - `ライター管理/月別消化率.md` に全社サマリーを記録

## 月次運用サイクル
| タイミング | 作業内容 | 更新ファイル |
|---|---|---|
| 月初 | 今月の発注予定を記入 | `クライアント/<社名>/07_ライター稼働/YYYY_MM_稼働.md` |
| 随時 | 納品・検収をステータス更新 | 同上 |
| 月末 | ①各案件の原価を確定 | 同上 |
| | ②案件別売上・経費を転記 | `収支管理/YYYY/01_収支.md` |
| | ③全社ライター稼働を集計 | `ライター管理/月別消化率.md` |
